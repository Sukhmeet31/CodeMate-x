# CodeMate-X

CodeMate-X is an AI-powered coding workspace that includes:

- A real-time code editor
- AI assistant panel
- Project dashboard
- Animated modern UI (Framer Motion + Tailwind)
- Firebase authentication (Email/Password)
- Chat interface
- Documentation section
- Project and analytics pages

## Tech Stack
- React (Vite)
- Tailwind CSS
- Framer Motion
- Firebase Authentication
- JavaScript

## Getting Started

### Install dependencies
npm install

### Run development server
npm start

### Build
npm run dev
